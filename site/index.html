🌑 The problem

You have 20 lint categories, 200 missing type annotations, and copy-pasted error handling. You know what to do — you just don't have the hours.

🌕 The solution

Write specs. Start the loop. Wake up to clean commits.

How It Works

1

Write a spec

Define requirements, shell-verifiable acceptance criteria, and exclusions.

2

Run owloop run

Ollie picks up your spec queue and starts the overnight loop.

3

Fresh iterations

Each task spawns a clean agent context. Ollie commits only when shell checks pass.

4

Review the report

Open the rich HTML report in the morning and merge the worktree.

See it in action

ollie@night:~
